# FoodApp

## Hướng dẫn sử dụng

1. Chuẩn bị môi trường

- Cài nodejs nếu chưa có
- Cài máy ảo
- Cài expo bằng lệnh: yarn global add expo-cli

2. Thêm thư viện

- Thêm thư viện bằng lệnh: yarn

3. Kết nối database

- Coppy file .env vào thư mục gốc của ứng dụng

4. Run app

- Chạy bằng lệnh: yarn start
